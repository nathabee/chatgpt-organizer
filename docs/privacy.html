ChatGPT Organizer – Privacy Policy

ChatGPT Organizer is a client-side browser extension. All functionality runs locally in the user’s browser.

Data usage

  • The extension accesses content on chatgpt.com only after explicit user interaction.
  • No data is transmitted to external servers.
  • No personal data is collected, stored, sold, or shared.
  • All processing happens locally in the browser.

Permissions

Requested browser permissions are used exclusively to provide the extension’s core functionality, such as organizing and managing the user’s own ChatGPT conversations.

No tracking

  • No analytics
  • No advertising
  • No tracking
  • No remote code execution

Support

This extension is provided as-is. Support is offered on a best-effort basis via GitHub Issues.

This privacy policy applies only to ChatGPT Organizer and does not cover ChatGPT or OpenAI services.
