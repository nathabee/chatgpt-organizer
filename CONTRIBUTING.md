# Contributing

Thanks for your interest in ChatGPT Organizer.

This project is currently maintained by a single author.
Contributions, bug reports, and feedback are welcome.

## How to contribute

- Open an issue for bugs, questions, or ideas
- Describe problems clearly and provide steps to reproduce
- Keep discussions technical and respectful

## Pull requests

If you want to submit code:
- open an issue first to discuss the change
- keep PRs small and focused
- follow the existing project structure

There is no formal roadmap yet.
